# Retrato dorado — Estética moderna y elegante

Sitio Next.js que presenta un retrato fotográfico con estilo hiperrealista y luz cálida de golden hour. Usa imágenes relevantes de Unsplash según palabras clave.

## Desarrollo

- `npm install`
- `npm run dev`

## Build

- `npm run build`
- `npm start`

## Despliegue (Vercel)

Este proyecto está preparado para desplegarse en Vercel.
